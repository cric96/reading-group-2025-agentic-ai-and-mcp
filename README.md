## How to run the presentation
To run the presentation you have two options:
### RISE (discouraged)
1. install the dependencies `uv pip install -e .`
2. run the jupyter notebook `jupyter notebook presentation.ipynb`
### nbconvert (suggested)
1. install the dependencies `uv pip install -e .`
2. run the script `python serve_slides.py`
3. open your browser to http://localhost:8000/presentation.slides.html
